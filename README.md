This repository is created to demonstrate how to pass parameters from a package to a file.

## Usage

To use this code, follow these steps:

1. Clone the repository to your local machine.
2. Open the terminal and navigate to the repository directory.
3. Open the file `index.js` and modify the parameters as needed.
4. Run the code by executing the following command:
    ```
    pnpm start
    ```

Feel free to explore the code and experiment with different parameters. Happy coding!
